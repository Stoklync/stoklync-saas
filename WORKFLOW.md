# Stoklync SaaS — Customer Workflow

Workflow for consultants and service businesses. Clear purpose at every step.

---

## 1. Get Started
- **Sign up** with email, name, company
- **Confirm email** (if required)
- **Sign in** → lands on Dashboard

## 2. Set Up Your Brand
- Add **logo**, **colors**, **contact info**
- Set **business type** (service / product)
- Saves to Website and CRM

## 3. Create Your Website
- **Live preview** iframe in dashboard (refresh after save)
- Edit **headline**, **about**, **contact form**
- **Testimonials** section on public site
- Publish → live at `/site/[orgId]`

## 4. Get Leads
- Contact form on your site
- Inquiries appear in **CRM** as leads
- Add leads manually (from calls, events, etc.)

## 5. Manage in CRM (3-pillar)
- **Consulting** pipeline: DISCOVERY → STRATEGY SESSION → PROPOSAL → CONTRACT SIGNED
- **Deals** pipeline: NEW → QUALIFIED → PROPOSAL → NEGOTIATION → WON/LOST
- **Product** (coming): orders, SKU, fulfillment
- **Services** (coming): projects, milestones, time tracking
- **Lead detail**: Contact 360 timeline, notes, **Create quote** (draft)
- **Real-time**: new leads appear instantly (notification dot)

## 6. Marketing (Communication)
- **Recipient picker**: send to one, some, or all leads
- **Lead sources**: website URL, webhook (Zapier/Facebook Lead Ads)
- **Automations** (coming): e.g. "Proposal 3+ days → follow-up email"

## 7. Support
- Tickets from website contact form
- Create tickets manually
- Assign, reply, resolve
- Reply via email (mailto)

---

## Dashboard Flow (What Users See)

| Step | Tab / Area | Purpose |
|------|------------|---------|
| 1 | Home | Welcome + getting started checklist |
| 2 | Website | View site, Branding, Contact, CMS |
| 3 | CRM | Pipeline, lead detail, notes, actions |
| 4 | Marketing | Compose & send emails to leads |
| 5 | Support | Tickets, create, assign, reply |
| 6 | Team | Invite, roles |
| 7 | Settings | Org, branding |

---

## First-Time User Path

1. Sign up → Dashboard
2. See "Getting started" (Step 1: Add branding)
3. Add logo, company name, emails
4. Go to Website → View your site
5. Share site → leads come in
6. CRM → see leads, add notes, follow up
7. Marketing → send bulk email
8. Support → handle tickets

---

## Principles

- **Simple** — No clutter. One main action per screen.
- **Purpose** — Every tab has a clear job.
- **Workflow** — Lead → Contact → Follow up → Close → Support.
